Visão Geral

Este projeto de Power BI foi desenvolvido para a empresa Franfoods com o objetivo de analisar o desempenho comercial, acompanhar vendas versus metas e fornecer insights claros para tomada de decisão gerencial.

O foco principal é transformar dados operacionais (vendas diárias e metas mensais) em informação estratégica, utilizando modelagem adequada, medidas em DAX e visualizações objetivas.
